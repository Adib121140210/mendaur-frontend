## ðŸ” TROUBLESHOOTING

### Problem: Bottom nav not showing
```
Solution:
1. Check window width: F12 > Console > window.innerWidth
2. Should be â‰¤1024px for nav to show
3. If still not visible:
   - Clear cache: Ctrl+Shift+Delete
   - Hard refresh: Ctrl+Shift+R
   - Check CSS loaded: F12 > Network > .css files
```

### Problem: Content hidden under nav
```
Solution:
1. This is already fixed in layout.css
2. If issue persists:
   - Clear cache: Ctrl+Shift+Delete
   - Hard refresh: Ctrl+Shift+R
   - Check @media queries applied
```

### Problem: Navigation not working
```
Solution:
1. Check console for errors: F12 > Console
2. Verify routes in App.jsx
3. Check that useNavigate hook is working
4. Verify URL changed after clicking
```

### Problem: Icons not showing
```
Solution:
1. Check SVG loaded: F12 > Console > check errors
2. Verify fill="currentColor" in SVG
3. Check CSS color applied
4. Hard refresh browser
```
